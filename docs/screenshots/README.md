## Screenshot Guidelines

- **Resolution**: 1920x1080 or similar
- **Format**: PNG for stills, GIF for animation
- **Quality**: High quality, no compression artifacts
- **Content**: Remove any personal information
- **Lighting**: Use dark theme (consistent with Scribe)
- **Crop**: Remove unnecessary screen space

## How to Take Screenshots

**Windows:**
- Use Windows Snipping Tool (Win+Shift+S)
- Or use ShareX for advanced features
- For GIF: Use ScreenToGif

**Tools for GIFs:**
- ScreenToGif (Windows): https://www.screentogif.com/
- LICEcap (Cross-platform): https://www.cockos.com/licecap/
- Kap (Mac): https://getkap.co/

## Editing Tips

- Crop to relevant content only
- Add subtle drop shadow if needed
- Ensure text is readable
- Highlight key features with arrows/circles (optional)
